Target JwtAuthGuard (src/shared/auth/jwt-auth.guard.ts) — pure CanActivate, NO Passport:
```typescript
import { CanActivate, ExecutionContext, Injectable, UnauthorizedException } from '@nestjs/common';
import { JwtService } from '@nestjs/jwt';
import { Request } from 'express';

@Injectable()
export class JwtAuthGuard implements CanActivate {
  constructor(private readonly jwtService: JwtService) {}

  async canActivate(context: ExecutionContext): Promise<boolean> {
    const request = context.switchToHttp().getRequest<Request>();
    const token = this.extractToken(request);
    if (!token) throw new UnauthorizedException('invalid_token');
    try {
      const payload = await this.jwtService.verifyAsync(token);
      // Attach normalized user to request: { sellerId, email }
      (request as any).user = { sellerId: payload.sub, email: payload.email };
      return true;
    } catch {
      throw new UnauthorizedException('invalid_token');
    }
  }

  private extractToken(request: Request): string | null {
    const header = request.headers.authorization;
    if (!header) return null;
    const [type, token] = header.split(' ');
    return type === 'Bearer' && token ? token : null;
  }
}
```

       import { JwtService } from '@nestjs/jwt';

       @Injectable()
       export class JwtAuthGuard implements CanActivate {
         constructor(private readonly jwtService: JwtService) {}

         async canActivate(context: ExecutionContext): Promise<boolean> {
           const request = context.switchToHttp().getRequest<Request>();
           const token = this.extractToken(request);
           if (!token) throw new UnauthorizedException('invalid_token');
           try {

             (request as any).user = { sellerId: payload.sub, email: payload.email };
             return true;
           } catch {

           }

         private extractToken(request: Request): string | null {
           const header = request.headers.authorization;
           if (!header) return null;
           const parts = header.split(' ');

           const [type, token] = parts;

             amount: txResponse.amount,

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Authorization header → JwtAuthGuard | Untrusted Bearer token crosses from HTTP into server; guard is the first auth checkpoint |
| ConfigService ← env / .env file | Env values loaded once at bootstrap; Joi validation rejects malformed/missing required values |
| HttpService → .NET service | Internal HTTP call; baseURL sourced from DOTNET_SERVICE_URL env |
| JwtService.verifyAsync | Validates signature + expiry using JWT_SECRET from ConfigService |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-11 | Spoofing | JwtAuthGuard Bearer token | mitigate | `JwtService.verifyAsync()` validates HS256 signature against JWT_SECRET; tokens without Bearer scheme or with invalid payload throw UnauthorizedException('invalid_token'). Unit test coverage forces all failure paths. |
| T-01-12 | Tampering | ConfigService.get('JWT_SECRET') | mitigate | Joi validationSchema marks JWT_SECRET as required() — app fails to boot if env var is missing or empty. AuthModule registerAsync reads secret once at module init; subsequent token verifications use the already-loaded secret (no re-read of env). |
| T-01-13 | Information Disclosure | PublicChargeService error path | mitigate | On HttpService failure, `PiiSanitizer.safeBody(payload)` invoked BEFORE throwing HttpException. Raw axios response body (which may echo PII) never reaches the HTTP response; only `{ error: 'payment_processor_unavailable', correlationId }` returned. |
| T-01-14 | Information Disclosure | JwtAuthGuard error message | mitigate | Only throws `UnauthorizedException('invalid_token')` — never reveals whether the failure was expiry, signature mismatch, or malformed header. Prevents token-probing oracles. |
